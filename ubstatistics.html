This is a plugin for Unity I developed to easily handle statistics, especially useful for RPGs!

The main scripts used are shown below and facilitates an easy way to create and add stats as well as view it's current value alongside it's max and min. It allows custom actions to occur when a change occurs on the stats value or when it reaches the max or min values. The user can also save default stats for use when creating a few different prefab types so you can save a set of stats out and load it onto another statistic container easily using the "Default" and "Set as Default" buttons. It also provides an easy way to test the networking of the stats with the "Network Update" button.

The network statistics script will automatically update networks when a stat changes so it updates on clients and server.
