--- layout: page title: ubDialogue ---

This is a plugin for Unity I developed to easily write dialogues.

I started the idea from seeing this thread on the Unity forums: simple node editor and used it as a jumping board to write my own dialogue editor.

The plugin also comes with a simple to use API to facilitate usage of it, you simply call void GetNextConnections(out List connections) and handle the choices if there are any or just display the first next text.
